# 高级文章生成器功能说明

## 🎯 概述

高级文章生成器是一个专业级的博客内容生成工具，提供9大设置模块和100+可调参数，让您完全掌控文章生成的每个细节。

## 📋 功能模块

### 1. 核心设置 (Core Settings)

**功能：** 配置文章的基本属性

**可配置项：**
- 文章主题（必填）
- 文章类型：
  - 博客文章
  - 教程指南 (How-to)
  - 列表文章 (Listicle)
  - 对比评测
  - 产品评测
  - 新闻报道
  - 案例研究
- AI模型选择：
  - Claude 3.5 Sonnet（推荐中文）
  - GPT-4 Turbo
- 文章数量：1-10篇
- 发布日期
- 目标国家/地区
- 目标语言：简体中文、英语、日语、韩语

**使用建议：**
- 选择合适的文章类型可以让AI生成更符合预期的内容结构
- 不同AI模型各有优势，Claude更适合中文，GPT-4在英文方面表现优秀

---

### 2. 品牌声音 (Brand Voice)

**功能：** 定制文章的语言风格和品牌特色

**可配置项：**
- 预设语言风格：
  - 专业正式
  - 轻松随意
  - 友好亲切
  - 技术性强
  - 对话式
  - 权威性
  - 热情洋溢
- 自定义品牌声音（可选）

**使用示例：**
```
自定义品牌声音示例：
"我们的品牌注重数据驱动的见解，使用简洁的语言，
避免行业术语，强调实用性和可操作性。"
```

**使用建议：**
- 结合预设风格和自定义描述可以获得最佳效果
- 自定义声音应该具体描述：避免什么、强调什么

---

### 3. 详细信息 (Details to Include)

**功能：** 添加必须在文章中包含的特定信息

**可配置项：**
- 额外信息文本框（支持多行）

**适合添加的内容：**
- 最新统计数据
- 产品特性和规格
- 公司里程碑
- 行业趋势
- 专家观点
- 案例数据

**使用建议：**
- 提供具体的数据和事实
- 包含来源链接（如果有）
- 避免过于笼统的描述

---

### 4. 媒体中心 (Media Hub)

**功能：** 配置文章中的图片和视频

**图片设置：**
- 数量：0-10张
- 尺寸：
  - 1920×1080（标准）
  - 1200×630（社交媒体）
  - 800×600（缩略图）
- 风格：
  - 专业商务
  - 现代简约
  - 创意艺术
  - 极简主义

**视频设置：**
- 是否包含视频建议

**使用建议：**
- 每1000字配1-2张图片效果最佳
- 社交媒体分享建议使用1200×630尺寸
- 图片风格应与品牌调性一致

---

### 5. SEO优化 (SEO)

**功能：** 优化文章的搜索引擎表现

**可配置项：**
- 目标关键词（必填，3-5个推荐）
- 焦点关键词（可选，主要优化目标）

**SEO建议：**
- ✓ 使用长尾关键词提高排名机会
- ✓ 关键词密度保持在0.5%-2.5%之间
- ✓ 在标题和前100字中包含主关键词

**使用建议：**
- 主关键词：搜索量较大的核心词
- 长尾关键词：更具体的3-5个词组合
- 焦点关键词：最希望排名的那个词

---

### 6. 文章结构 (Structure)

**功能：** 自定义文章的组织结构

**可配置项：**
- 引言简介要求
- 主要章节（H2标题）
- 子章节（H3标题）
- 是否包含列表和要点
- 结论总结要求

**使用示例：**
```
主要章节（H2）：
1. 什么是AI内容生成
2. AI内容生成的优势
3. 如何选择AI工具
4. 最佳实践和技巧

子章节（H3）：
- 成本节约
- 时间效率
- 质量保证
```

**使用建议：**
- H2章节：3-6个为佳
- 每个H2下建议有2-3个H3
- 列表可以提高可读性

---

### 7. 链接设置 (Linking)

**功能：** 管理内部和外部链接

**内部链接：**
- 添加站内相关文章URL
- 建议2-5个

**外部链接：**
- 链接类型：
  - 自动添加权威来源
  - 手动指定
  - 不添加外部链接
- 备注说明

**使用建议：**
- 内部链接增强网站结构，提升SEO
- 外部链接应指向高权威域名
- 避免链接到竞争对手

---

### 8. 联网搜索 (Connect to Web)

**功能：** 从网络获取最新信息

**搜索深度选项：**
- **轻度**：快速概览（1-2分钟）
- **中度**：平衡速度与深度（3-5分钟）
- **深度**：全面研究（5-10分钟）

**获取的信息类型：**
- 最新统计数据
- 行业趋势
- 新闻动态
- 专家观点
- 案例研究

**使用建议：**
- 时效性强的话题选择深度搜索
- 常规内容选择中度即可
- 快速测试可用轻度

---

### 9. 社交发布 (Syndication)

**功能：** 同步发布到社交媒体平台

**支持平台：**
- 🐦 Twitter / X
- 💼 LinkedIn
- 👥 Facebook
- 📧 Email Newsletter
- 📌 Pinterest

**功能说明：**
- 自动生成适合各平台的摘要
- 优化图片尺寸
- 添加话题标签建议

**使用建议：**
- B2B内容重点选LinkedIn
- B2C内容重点选Facebook和Pinterest
- 专业内容建议Twitter和LinkedIn

## 🎯 最佳实践

### 1. 关键词策略
- 主关键词1个
- 次要关键词2-3个
- 长尾关键词1-2个

### 2. 内容长度
- 博客文章：1500-2000字
- 教程指南：2000-2500字
- 列表文章：1200-1800字
- 深度分析：2500-3500字

### 3. 结构规划
- H1标题：1个
- H2章节：3-6个
- H3子章节：每个H2下2-3个
- 列表：适当使用

### 4. 媒体配置
- 封面图：1张
- 内容图：每1000字1-2张
- 图表：复杂内容使用

### 5. 链接策略
- 内部链接：2-5个
- 外部链接：2-3个权威来源
- 锚文本：自然相关
