## 🎛️ 侧边栏功能详解

### Tab A: ⚙️ 文章设置

#### 1. 状态管理
```
草稿 ➜ 待审核 ➜ 已发布
```
- **草稿**：默认状态，未公开
- **待审核**：提交审核
- **已发布**：对外可见

#### 2. 作者信息
- 输入作者名称
- 用于Schema标记
- 影响E-E-A-T权威性评分

#### 3. 发布排期
- 选择发布日期和时间
- 支持定时发布
- 自动更新sitemap

#### 4. 特色图片
**上传步骤：**
1. 点击虚线框区域
2. 选择图片文件
3. 图片自动上传并预览
4. **必填**：添加Alt文本描述

**Alt文本要求：**
- 准确描述图片内容
- 包含相关关键词
- 长度建议10-50字符

#### 5. 分类与标签
- **分类**：选择主要类别
- **标签**：用逗号分隔，例如：`SEO, 内容营销, AI`

### Tab B: 🔍 SEO配置

#### 1. SERP预览 🔍
**实时显示您的文章在Google搜索结果中的样子**

```
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
您的SEO标题会显示在这里
https://yoursite.com/article-slug
您的元描述会显示在这里，吸引用户点击...
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
```

#### 2. URL Slug
**最佳实践：**
- 使用小写字母
- 用连字符分隔单词
- 包含主关键词
- 保持简短（3-5个词）

**示例：**
```
❌ bad: Article_Title_2024
✅ good: seo-article-optimization
```

#### 3. SEO标题
**长度指南：**
- ⚠️ < 30字符：过短
- ✅ 30-60字符：理想
- ⚠️ > 60字符：可能被截断

**优化技巧：**
- 前置核心关键词
- 包含品牌名
- 吸引点击
- 避免关键词堆砌

**示例：**
```
❌ 不佳：文章
❌ 过长：这是一篇关于如何使用AI工具进行SEO优化的完整指南包含所有细节
✅ 理想：AI SEO优化完整指南 | 品牌名
```

#### 4. 元描述
**长度指南：**
- ⚠️ < 120字符：过短
- ✅ 120-160字符：理想
- ⚠️ > 160字符：可能被截断

**写作要点：**
- 概括文章价值
- 包含焦点关键词
- 添加行动号召
- 吸引点击

**示例：**
```
❌ 不佳：这是一篇文章。
✅ 理想：学习如何使用AI工具优化SEO，提升排名。包含10个实用技巧和案例分析。立即阅读！
```

#### 5. 焦点关键词
**作用：**
- 主要优化目标
- 触发密度检查
- 影响所有SEO评分

**选择标准：**
- 搜索量适中
- 竞争度可控
- 与内容高度相关

#### 6. 相关关键词
**建议数量：** 3-5个

**类型：**
- 主关键词的变体
- 相关长尾词
- LSI（潜在语义）关键词

#### 7. 高级选项 ⚙️

**Canonical链接**
- 用于避免重复内容
- 指向原始发布URL
- 适用于转载/聚合场景

**Robots指令**
- **Index**: 允许搜索引擎索引
- **Follow**: 允许跟踪页面链接
- 默认两者都开启

**Schema架构类型**
选择最符合内容的类型：
- **Article** - 通用文章
- **BlogPosting** - 博客文章
- **NewsArticle** - 新闻报道
- **Review** - 评测文章
- **HowTo** - 教程指南
- **FAQ** - 问答文章

### Tab C: 📊 审计与评分

#### 1. 综合得分面板

```
┌────────────────┐
│   综合得分      │
│     85        │
│    优秀       │
└────────────────┘
```

**评分标准：**
- **80-100**: 🟢 优秀 - 可以发布
- **60-79**: 🟡 良好 - 建议优化
- **0-59**: 🔴 需改进 - 必须优化

#### 2. E-E-A-T评分

Google质量评估四要素：

**Experience (经验)**
- 是否包含实践案例
- 是否有亲身经历
- 关键词：`我的经验`、`实践`、`测试`

**Expertise (专业)**
- 是否有专业术语
- 是否引用数据
- 关键词：`研究`、`分析`、`技术`

**Authority (权威)**
- 是否有外部引用
- 是否有权威链接
- 是否有作者信息

**Trustworthiness (可信)**
- 是否有可信来源
- 是否有事实依据
- 是否有官方引用

#### 3. SEO检查清单

**实时检查的10个项目：**

| 检查项 | 标准 | 图标 |
|-------|------|------|
| H1标签唯一 | 有且仅有1个H1 | ✅/❌ |
| SEO标题长度 | 30-60字符 | ✅/⚠️/❌ |
| 元描述长度 | 120-160字符 | ✅/⚠️/❌ |
| 关键词在标题中 | 焦点词出现在标题 | ✅/❌ |
| 关键词在描述中 | 焦点词出现在描述 | ✅/⚠️ |
| 关键词密度 | 0.5%-2.5% | ✅/⚠️/❌ |
| 图片Alt标签 | 所有图片有Alt | ✅/⚠️/❌ |
| 内部链接 | ≥2个 | ✅/⚠️/❌ |
| 外部链接 | ≥1个 | ✅/⚠️ |
| 内容长度 | ≥1500字 | ✅/⚠️/❌ |

**图标含义：**
- ✅ **通过** - 符合最佳实践
- ⚠️ **警告** - 建议改进
- ❌ **失败** - 需要修复

#### 4. 优化建议

系统根据检查结果自动生成改进建议：

**示例建议：**
```
🔵 关键词密度过低（0.3%）
   建议：增加"AI工具"的使用频率，目标1.5%

🔵 缺少外部链接
   建议：添加至少1个权威来源的引用链接

🔵 文章长度不足（800字）
   建议：扩充内容至至少1500字
```

## 🎨 最佳实践工作流

### 第一步：框架搭建
1. 输入吸引人的标题
2. 添加H2主要章节（3-6个）
3. 每个H2下添加H3子章节（2-3个）

### 第二步：内容填充
1. 写引言（100-150字）
2. 填充各章节内容
3. 添加列表和引用
4. 插入相关图片

### 第三步：SEO优化
1. 切换到"SEO配置"Tab
2. 设置焦点关键词
3. 优化SEO标题和描述
4. 检查SERP预览效果

### 第四步：质量检查
1. 切换到"审计与评分"Tab
2. 查看综合得分（目标≥80）
3. 修复所有❌项
4. 优化⚠️项

### 第五步：最终调整
1. 切换到"文章设置"Tab
2. 上传特色图片+Alt文本
3. 设置分类和标签
4. 选择作者和发布时间

### 第六步：发布
1. 最后检查SEO评分
2. 预览桌面和移动效果
3. 点击"发布"按钮

## 💡 SEO优化技巧

### 关键词优化
```
✅ DO:
- 在标题前半部分包含焦点关键词
- 自然融入正文，密度1.5%左右
- 使用关键词变体和同义词

❌ DON'T:
- 关键词堆砌（密度>2.5%）
- 强行插入关键词
- 忽略用户体验
```

### 内容结构
```
理想文章结构：
H1: 主标题
  ├─ 引言 (100-150字)
  ├─ H2: 章节1
  │   ├─ H3: 小节1.1
  │   └─ H3: 小节1.2
  ├─ H2: 章节2
  │   ├─ H3: 小节2.1
  │   └─ H3: 小节2.2
  ├─ H2: 章节3
  └─ 结论 (100-150字)
```

### 链接策略
**内部链接：**
- 链接到相关文章
- 使用描述性锚文本
- 避免"点击这里"

**外部链接：**
- 引用权威来源
- 添加rel="nofollow"（如需）
- 定期检查链接有效性

### 图片优化
```
最佳实践：
1. 文件名：seo-optimization-guide.jpg ✅
   避免：IMG_1234.jpg ❌

2. Alt文本：SEO优化完整指南配图 ✅
   避免：图片 ❌

3. 文件大小：< 500KB
4. 格式：WebP > JPG > PNG
```

## 🐛 常见问题

### Q: 为什么我的关键词密度总是不合格？
**A:**
- 焦点关键词应该是短语，不是单词
- 自然融入，避免刻意插入
- 使用变体和同义词
- 目标密度：1.5%左右

### Q: SEO标题总是超出60字符怎么办？
**A:**
- 删除无关紧要的词
- 使用缩写（SEO vs 搜索引擎优化）
- 移除品牌名（放在末尾）
- 标题应精炼，不是句子

### Q: 什么是SERP预览？
**A:**
SERP = Search Engine Results Page（搜索引擎结果页）
预览显示您的文章在Google搜索中的样子。

### Q: 为什么需要Alt文本？
**A:**
1. **SEO**：搜索引擎靠Alt理解图片
2. **可访问性**：屏幕阅读器为盲人读取
3. **备用**：图片加载失败时显示

### Q: 草稿会被搜索引擎索引吗？
**A:**
不会。只有状态为"已发布"的文章才会：
- 出现在sitemap
- 被搜索引擎索引
- 对外可见
